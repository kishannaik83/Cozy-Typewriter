# Cozy Typewriter

A quiet, meditative web experience where your moods become words, typed out on a virtual vintage typewriter. 

Designed for moments of pause, reflection, and gentle inspiration.

## Features
- **Meditative Interface**: Clean, distraction-free design with a cozy parchment aesthetic.
- **Mood-Based Quotes**: Select from 6 unique moods (Happy, Sad, Calm, Motivated, Thoughtful, Lonely) to receive tailored quotes.
- **Authentic Typewriter Simulation**: Features realistic key-press sounds, carriage returns, and vibrating animations.
- **High-Quality Sharing**: Export your quotes as high-resolution 9:16 cards perfect for Instagram Stories.
- **Responsive Design**: Flawless experience across desktop, tablet, and mobile (iOS & Android).
- **Offline Capable**: Fast-loading assets and optimized performance.

## Tech Stack
- **Frontend**: React (TypeScript)
- **Styling**: Vanilla CSS (Modern CSS variables + custom design system)
- **Asset Processing**: html2canvas (Dynamic image generation)
- **Deployment**: Optimized for Vercel/Vite

## Local Development
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Deployment
This project is configured for one-click deployment to Vercel. 
- Build Command: `npm run build`
- Output Directory: `dist`
- Framework Preset: Vite
